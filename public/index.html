This dashboard dynamically reads from the Isolation Forest Machine Learning model. The Nginx server acts as a decoy target. When the system detects a burst of random port probes, the features matrix deviates from standard HTTP baselines, triggering an anomaly alert.

How it Works: Isolation Forest

Unlike traditional IDS that use known "signatures," this system uses unsupervised learning to isolate outliers.

  • Isolation: Anomalies (like port scans) are rare and different, making them easier to "isolate" using fewer random partitions than normal data points.
  • No Labels Needed: The model learns the "normal" traffic shape automatically, allowing it to detect new, unknown attack patterns.
